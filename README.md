💊 Pharmaceutical Sales Analysis (EDA Project)

📌 About This Project

I worked on this pharmaceutical sales dataset as part of my data analytics practice. The main goal was to understand how sales, customers, and products are performing and to find ways to improve business profit using data.

This is one of my first end-to-end EDA projects using Python.

🎯 What I Tried to Do

* Understand which cities generate more sales
* Find top customers who contribute more revenue
* Analyze which products are profitable
* Identify any loss-making products
* Check how sales representatives are performing


🛠️ Tools I Used

* Python
* Pandas
* Matplotlib
* VS Code

📊 About the Dataset

The dataset includes:

* Customer details (name, city, country)
* Product details (product name, class)
* Sales data (quantity, price, total sales)
* Sales team details (sales rep, manager, team)
* Time data (month, year)

🔍 What I Did

✔️ Data Cleaning

* Removed duplicate records
* Handled missing values
* Checked data types

### ✔️ Feature Creation

Created a new column Profit = Sales - (Quantity × Price)

✔️ Analysis

* City-wise Sales → Found which cities bring more revenue
* Customer Analysis → Identified high-value customers
* Product Analysis → Checked top-performing and low-performing products
* Sales Rep Performance → Compared performance of sales representatives
* Loss Analysis → Found products that are causing losses

📈 Visualizations

* Bar chart for top cities by sales
* Line chart for monthly sales trend
* Bar chart for top products

---

💡 Key Insights

* A few cities contribute most of the total sales
* Some customers generate a large portion of revenue
* Not all products are equally profitable
* There are some products that result in losses
* Sales performance differs between sales representatives

🚀 What I Suggest (Based on Data)

* Focus more on high-value customers
* Increase stock of profitable products
* Reduce or review loss-making products
* Improve performance of low-performing sales reps

📚 What I Learned

* How to clean and prepare real-world data using Pandas
* How to analyze customer and product performance
* How to calculate profit and identify business problems
* How to convert data into simple business insights

Final Note:

This project helped me understand how data analysis can be used in real business scenarios. I focused more on gaining insights rather than just creating charts.

I am continuing to improve my skills in data analytics and visualization.
